## functions
- /start
- /add [item]
- /delete [index]
- /peek
- / clear
 
## getting API keys
- [Telegram Bot key](https://core.telegram.org/bots#3-how-do-i-create-a-bot)
  + update token information in `firebase.json`
- [Firebase key](https://console.firebase.google.com/)
  + update Firebase information in `app.js`

## run on local server
```
npm start
```
```
node app.js
```
