## getting API keys
- [Telegram Bot key](https://core.telegram.org/bots#3-how-do-i-create-a-bot)
  + update information in `firebase.json`
- [Firebase key](https://console.firebase.google.com/)
  + update information in `app.js`

## run on local server
```
npm start
```
```
node app.js
```
